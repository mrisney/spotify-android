Introduction

This is a simple caching application based on an extensible pattern of cache eviction based on the caching library developed in this project : Caching Algorithim.

The sample android application allows the user to search Google Images for a search term. Using the results, and parsing them with JSoup, a collection of Key Values are placed into the cache as ByteBuffer.
You can browse the javadocs for more information.
